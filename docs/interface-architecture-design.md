# 《独家童话》界面架构设计

> 文档说明：本文档用于定义《独家童话》的页面架构、主 Tab 结构、功能子页面、页面流程关系和特殊交互页面，是后续补齐路由、检查页面覆盖度和规划页面栈的基础依据。

## 一、核心界面（Tab 导航）

### 1. 首页 - 记录中心

- 今日记录快捷入口
- 最近记录列表（日记 + 照片混合流）
- 快速创建按钮（日记 / 照片 / AI 创作）

### 2. 情侣空间

- 双方动态流（时间轴展示）
- 互动功能区（点赞、评论）
- 恋爱天数计数器

### 3. AI 创作工坊

- AI 漫画生成入口
- AI 短视频生成入口
- 创作历史展示

### 4. 我的

- 个人信息
- 纪念日管理
- 数据导出 / 备份
- 设置

## 二、功能子页面

### 账号与关联

- 登录 / 授权页
- 情侣邀请页（生成邀请码 / 二维码）
- 情侣绑定确认页
- 情侣信息设置页

### 记录相关

- 日记编辑器（富文本编辑）
- 日记详情页
- 照片上传页
- 相册浏览页（网格 / 时间线视图）
- 时光胶囊设置页
- 标签管理页

### AI 创作相关

- AI 漫画生成配置页（风格选择、人设管理）
- 文本转漫画页
- 照片转漫画页
- AI 短视频配置页（音乐、字幕、转场）
- 视频预览 / 编辑页
- 生成进度页

### 互动相关

- 情侣动态详情页
- 评论列表页
- 互动通知页

### 纪念日相关

- 纪念日列表页
- 纪念日添加 / 编辑页
- 纪念日倒计时页
- 纪念日专属记录模板页

### 数据管理

- PDF 导出配置页
- 导出预览页
- 数据备份 / 恢复页
- 存储空间管理页

## 三、页面流程关系

```text
登录页 -> 判断是否绑定情侣
  ├─ 未绑定 -> 邀请 / 被邀请流程 -> 绑定成功 -> 进入主界面
  └─ 已绑定 -> 直接进入主界面（Tab 导航）
```

主界面 Tab 结构：

```text
主界面
  ├─ Tab1: 首页
  ├─ Tab2: 情侣空间
  ├─ Tab3: AI 创作
  └─ Tab4: 我的
```

各功能模块使用独立页面栈，并支持深层导航。

## 四、特殊交互页面

- 引导页（首次使用）
- 加载动画页（AI 生成时）
- 草稿箱
- 搜索页（记录搜索）
- 分享预览页
- 会员权益说明页
- 帮助与反馈页
