# 《独家童话》UI 设计总方向

## 一、产品气质定位

关键词：

- 童话绘本
- 温柔治愈
- 纸张感
- 手写感
- 收藏感
- 私密浪漫
- 回忆沉淀
- AI魔法感（不是科技感）

注意：

❌ 不要做成：

- 粉色社交 APP
- 低幼卡通
- 花里胡哨二次元
- 强科技 AI 风

✅ 应该像：

- 一本会呼吸的恋爱故事书
- 一本被时间慢慢写满的绘本日记

------

# 二、视觉风格（核心）

你提到中国色，这是非常正确的方向。

我建议不要用“高饱和少女粉”。

而是：

# 主色方案（中国色体系）

## 主品牌色

### 桃夭（主色）

一种偏柔和的桃粉。

适合：

- 主按钮
- Tab 激活
- AI按钮
- 重点强调

视觉气质：
 温柔、亲密、恋爱感。

------

## 藕丝秋半（辅助粉）

用于：

- 卡片背景
- 输入框
- Tag
- 弱强调

会让界面非常“奶油纸”。

------

## 枯玫瑰（高级感）

用于：

- 标题
- 重要文字
- 图标

这个颜色会直接拉高级感。

------

## 可可棕 / 棕茶

用于：

- 正文
- 边框
- 手绘线条

这是整个“绘本感”的灵魂。

------

## 月白（大面积背景）

用于：

- 页面底色
- 留白区域

不要纯白。

------

# 三、整体 UI 关键词

## 1. 大留白

不要信息爆炸。

页面呼吸感非常重要。

------

## 2. 卡片像贴纸

所有模块：

- 微阴影
- 圆角偏大
- 轻描边
- 像贴在纸上的照片

------

## 3. 手绘元素

推荐：

- 手绘爱心
- 星星
- 胶带
- 曲线
- 小花
- 纸张边缘

但：
 一定克制。

否则会廉价。

------

## 4. 动效

建议：

- 页面切换像翻书
- AI生成像“墨水扩散”
- 卡片展开像纸片弹起

不要：
 赛博粒子特效。

------

# 四、字体建议

## 中文

推荐：

- 霞鹜文楷（标题）
- HarmonyOS Sans（正文）
- 阿里巴巴普惠体（备选）

标题可以有一点手写味。

正文必须清晰。

------

# 五、Tab 导航设计（非常关键）

你的 Tab 非常合理。

我建议：

| Tab      | 气质     |
| -------- | -------- |
| 首页     | 日记本   |
| 情侣空间 | 双人宇宙 |
| AI创作   | 魔法工坊 |
| 我的     | 收藏册   |

注意：

不要叫“AI创作”。

建议改：

# 「童话工坊」

会高级很多。

------

# 六、首页（最核心页面）

建议结构：

## 顶部

- 今日日期
- 恋爱第 XXX 天
- 一句今日情绪文案

例如：

> “今天也被好好爱着。”

------

## 中部

### 今日记录入口

像：
 “翻开的日记页”。

按钮：

- 写日记
- 上传照片
- AI生成

------

## 下方

时间流卡片：

- 照片
- 日记
- 小贴纸
- 心情

混合流展示。

不要传统列表。

应该像：

# “回忆碎片墙”

------

# 七、情侣空间（重点差异化）

这个页面不要做朋友圈。

建议：

# 双人时间轴

比如：

- 她写了日记
- 你上传了照片
- AI生成了漫画
- 新纪念日

以“故事推进”形式展示。

像：

> 第 214 天 · 一起吃火锅

而不是：

“用户A发布动态”

------

# 八、AI功能（产品亮点）

这是你未来最容易爆的地方。

重点：

# AI不要做工具感

而是：

# “魔法感”

例如：

## 文本转漫画

不要：

“开始生成”

改成：

> “把回忆变成童话”

------

## 视频生成

不要：

“AI视频”

改成：

> “回忆放映机”

------

# 九、你最需要避免的问题

## 1. 页面太粉

情侣 APP 最大问题：
 土。

高级感来自：

- 克制
- 留白
- 低饱和
- 材质感

------

## 2. AI页面太科技

你的 AI 是“情感 AI”。

不是生产力 AI。

------

## 3. 图太多

你已经意识到了。

非常正确。

建议：

- 多图标
- 少Banner
- 少大图

否则会像模板站。

------

# 十、我建议的设计系统（重要）

你这个项目：
 一定要先做 Design System。

否则 30+ 页面一定会乱。

至少先统一：

## 基础组件

- 按钮
- 卡片
- 标签
- 输入框
- 弹窗
- 时间轴
- 空状态
- AI生成状态

------

## 插画系统

统一：

- 线条粗细
- 圆角
- 阴影
- 手绘风格

------

# 十一、推荐的页面优先级（非常重要）

不要一次做 30 页。

正确顺序：

# 第一阶段（MVP）

先做：

1. 登录
2. 首页
3. 写日记
4. 情侣空间
5. AI漫画入口
6. 我的
7. 纪念日

只需要 7 页。

就能形成完整产品闭环。

------

# 第二阶段

再补：

- AI视频
- PDF导出
- 时光胶囊
- 搜索
- 分享

------
