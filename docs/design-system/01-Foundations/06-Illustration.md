# Fairy 插画与角色系统

> 插画是《独家童话》品牌差异化的核心。每一个角色都是童话世界中的居民，它们让 App 从"工具"变成"伙伴"。

---

## 1. 设计哲学

### 1.1 插画情绪

- **手绘感**：像彩色铅笔在纸上涂色，有笔触纹理
- **Q版可爱**：大头小身，比例约 1:1.5
- **温暖柔和**：色彩限制在 Fairy 色板内，拒绝高饱和
- **有故事**：每个角色都有性格、有情绪、有使命

### 1.2 风格关键词

```
彩色铅笔 · 水彩晕染 · 圆润线条 · 大眼睛 · 小短腿 · 表情丰富 · 简单背景
```

## 2. 角色设定

### 2.1 角色总览

| 角色 | 英文名 | 使命 | 主色 | 性格 |
|------|--------|------|------|------|
| **小信使** | Messenger | 传递消息、通知 | 天空蓝 + 白色 | 活泼、有点迷糊 |
| **魔法画笔** | MagicBrush | AI 创作、生成 | 桃夭粉 + 金色 | 神秘、有创造力 |
| **伤心小熊** | SadBear | 错误、删除、安慰 | 可可棕 + 藕丝 | 温柔、容易受伤 |
| **独角兽** | Unicorn | 新功能、彩蛋、VIP | 彩虹渐变（柔和版）| 神秘、高贵 |
| **小精灵** | Fairy | 成功、收藏、庆祝 | 薄荷绿 + 金色 | 快乐、爱撒星星 |

### 2.2 小信使（Messenger）

```
外观：
  - 体型：小鸟 + 信封的结合体
  - 颜色：天空蓝身体，白色翅膀
  - 特征：头顶有一根翘起的羽毛，背着小信封

表情库：
  - 开心：翅膀张开，眼睛弯弯
  - 迷糊：转圈圈，虚线轨迹
  - 疲惫：翅膀下垂，喘气
  - 惊喜：眼睛变大，信封掉落

使用场景：
  - 网络错误："小信使迷路了，再试一次吧~"
  - 消息通知：小信使敲门动画
  - 加载中：小信使飞行轨迹
```

### 2.3 魔法画笔（MagicBrush）

```
外观：
  - 体型：一支有生命的画笔
  - 颜色：桃夭粉笔杆，金色笔尖
  - 特征：笔尖会发光，笔杆上有小星星装饰

表情库：
  - 创作中：笔尖发光，颜料飞溅
  - 完成：画出一个爱心
  - 思考：笔尖画问号
  - 疲惫：笔尖墨水干涸

使用场景：
  - AI 漫画生成："魔法画笔正在悄悄涂色..."
  - AI 视频生成："时光画笔正在绘制回忆..."
  - 加载状态：画笔旋转 + 星光粒子
```

### 2.4 伤心小熊（SadBear）

```
外观：
  - 体型：圆滚滚的小熊，约 2 头身
  - 颜色：可可棕身体，藕丝色肚子
  - 特征：大眼睛，小鼻子，耳朵圆圆的

表情库：
  - 伤心：眼泪汪汪，抱着膝盖
  - 安慰：拍拍你的肩膀，递纸巾
  - 害怕：缩成一团，发抖
  - 恢复：擦干眼泪，微笑

使用场景：
  - 删除确认："要把这段回忆藏进小黑屋吗？"
  - 网络错误："小信使迷路了..."
  - 空状态："这里还没有故事呢..."
```

### 2.5 独角兽（Unicorn）

```
外观：
  - 体型：小马，约 2 头身
  - 颜色：白色身体，彩虹色鬃毛（柔和版）
  - 特征：金色角，翅膀（可选），蹄子有星星

表情库：
  - 神秘：角发光，眼睛半闭
  - 惊喜：角闪烁，前蹄抬起
  - 睡觉：角变暗，Zzz
  - 孵化：在蛋壳里，蛋壳摇晃

使用场景：
  - 功能未开放："独角兽还在孵化这个神秘功能哦~"
  - VIP 入口：独角兽守门
  - 彩蛋发现：独角兽撒彩虹
```

### 2.6 小精灵（Fairy）

```
外观：
  - 体型：小人，约 2.5 头身
  - 颜色：薄荷绿裙子，金色头发
  - 特征：透明翅膀，手持魔法棒，脚不落地

表情库：
  - 开心：撒星星，转圈圈
  - 庆祝：放烟花，举奖杯
  - 害羞：脸红，躲到花后面
  - 认真：魔法棒发光，专注施法

使用场景：
  - 保存成功："回忆已收藏进故事书~"
  - 收藏动画：小精灵把回忆放进书架
  - 成就达成：小精灵颁发勋章
```

## 3. 插画风格指南

### 3.1 线条规范

| 属性 | 规范 | 说明 |
|------|------|------|
| 线条粗细 | 2-3px | 外轮廓 3px，内部细节 2px |
| 线条颜色 | 可可棕 80% | 柔和，不刺眼 |
| 端点 | 圆角 | 保持圆润感 |
| 断点 | 允许 | 增加手绘感 |

### 3.2 色彩规范

| 规则 | 规范 |
|------|------|
| 主色限制 | 必须使用 Fairy 色板 |
| 阴影 | 不使用黑色阴影，使用同色系的深色 |
| 高光 | 白色或奶油色，柔和过渡 |
| 渐变 | 允许轻微渐变，但必须柔和 |
| 背景 | 简单几何或留白，不复杂 |

### 3.3 比例规范

| 元素 | 比例 |
|------|------|
| 头 : 身 | 1 : 1.5（Q版）|
| 眼睛 | 占脸部 30% |
| 四肢 | 短粗，约 0.5 头长 |
| 装饰物 | 不超过角色体积的 20% |

## 4. 场景规范

### 4.1 空状态（Empty State）

```
构图：
  - 角色居中偏上（占画面 40%）
  - 标题在角色下方（h2, 20px）
  - 描述在标题下方（body, 14px, secondary）
  - 操作按钮在底部

动画：
  - 角色轻微上下浮动（呼吸感）
  - 偶尔眨眼
  - 空状态出现时：角色从下方滑入

示例：
  - 无日记：伤心小熊 + "这里还没有故事呢，来写第一篇吧~"
  - 无网络：小信使迷路 + "小信使迷路了，再试一次吧~"
  - 无收藏：小精灵 + "书架还是空的，去收藏一些回忆吧~"
```

### 4.2 加载状态（Loading State）

```
构图：
  - 角色居中（占画面 30%）
  - 故事化文案在角色下方
  - 进度条在文案下方（可选）

动画：
  - 角色循环动画（根据角色特性）
  - 文案逐字显示或循环切换
  - 进度条像墨水晕染

示例：
  - AI 生成：魔法画笔旋转 + "魔法画笔正在悄悄涂色..."
  - 网络请求：小信使飞行 + "小信使正在赶路..."
  - 保存中：小精灵装订 + "回忆正在藏进故事书..."
```

### 4.3 错误状态（Error State）

```
构图：
  - 角色居中（占画面 35%）
  - 错误标题（h2, 20px）
  - 安慰文案（body, 14px, secondary）
  - 重试按钮

动画：
  - 角色伤心表情
  - 轻微摇晃（-3° ~ 3°）
  - 眼泪动画（可选）

示例：
  - 网络错误：伤心小熊 + "小信使迷路了，再试一次吧~"
  - 生成失败：魔法画笔干涸 + "画笔累了，休息一会再试吧~"
```

### 4.4 成功状态（Success State）

```
构图：
  - 角色居中（占画面 35%）
  - 庆祝文案（h2, 20px, accent）
  - 描述文案（body, 14px）

动画：
  - 角色庆祝动作
  - 星星/爱心粒子从角色向外扩散
  - 文案淡入

示例：
  - 保存成功：小精灵撒星星 + "回忆已收藏进故事书~"
  - 生成完成：魔法画笔画爱心 + "你的童话画好了~"
```

## 5. 插画使用规则

### 5.1 尺寸规范

| 场景 | 尺寸 | 说明 |
|------|------|------|
| 空状态 | 120-160px | 足够大，有存在感 |
| 加载状态 | 80-120px | 适中，不喧宾夺主 |
| 错误状态 | 100-140px | 明显，引起注意 |
| 成功状态 | 100-140px | 庆祝感 |
| 导航图标 | 24-32px | 小尺寸，简洁 |
| 徽章/标签 | 16-20px | 极小，点缀 |

### 5.2 动画规范

| 场景 | 动画 | 时长 |
|------|------|------|
| 出现 | scale 0 → 1 + opacity 0 → 1 | 300ms, spring |
| 消失 | scale 1 → 0.9 + opacity 1 → 0 | 200ms, accelerate |
| 呼吸 | translateY 上下浮动 4px | 2000ms, linear loop |
| 眨眼 | scaleY 1 → 0.1 → 1 | 150ms, standard |
| 撒星星 | 粒子从中心向外扩散 | 400ms, decelerate |

## 6. AI 生成规则

1. **角色优先**：有对应角色场景时，必须使用角色插画
2. **风格统一**：所有插画必须像出自同一支画笔
3. **色彩限制**：必须使用 Fairy 色板，禁止高饱和
4. **比例一致**：Q版比例，大头小身
5. **表情丰富**：每个角色必须有多种表情状态
6. **场景完整**：空/加载/错误/成功状态必须有对应角色
7. **禁止**：写实风格、3D 渲染、照片、复杂背景
