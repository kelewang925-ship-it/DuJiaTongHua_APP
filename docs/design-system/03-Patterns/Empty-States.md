# Empty State Patterns

> 空状态模式。像翻开一本空白的故事书，温柔引导，不让人失落。

---

## 1. 空状态结构

```
EmptyState
├── 角色插画（120-160px，呼吸动画）
├── 标题（h2, 20px, semibold）
├── 描述（body, 14px, secondary）
└── 操作按钮（FairyButton，可选）
```

---

## 3. 动画规范

### 3.1 出现动画

```
角色：
  - translateY: 20px → 0
  - opacity: 0 → 1
  - duration: 400ms
  - easing: ease-spring

文案：
  - opacity: 0 → 1
  - duration: 300ms
  - delay: 200ms

按钮：
  - opacity: 0 → 1
  - duration: 300ms
  - delay: 400ms
```

### 3.2 呼吸动画

```
角色：
  - translateY: 0 → -4px → 0
  - duration: 2000ms
  - easing: ease-in-out
  - loop: infinite
```

---

## 4. AI 生成规则

1. 空状态必须有角色插画，禁止纯文字
2. 文案必须安慰 + 引导，禁止 "暂无数据"
3. 尽可能提供操作按钮，引导用户行动
4. 角色动画必须有呼吸感，不僵硬
