# Form Patterns

> 表单模式。像在手账本上填写，温柔引导，减少焦虑。

---

## 1. 表单结构

```
Form
├── 表单标题（可选）
├── 表单描述（可选）
├── 字段组
│   ├── 标签
│   ├── 输入框（FairyInput）
│   ├── 辅助说明
│   └── 错误提示
├── 字段组...
├── 操作按钮组
│   ├── 主要操作（FairyButton primary）
│   └── 次要操作（FairyButton ghost）
└── 底部提示（可选）
```

---

## 2. 字段规范

### 2.1 标签

```
标签名 *
```

- 字号：14px，medium
- 颜色：text-primary
- 必填：红色星号 *（state-error）
- 间距：标签与输入框 4px

### 2.2 输入框

- 使用 FairyInput 组件
- 高度：48px
- 圆角：8px
- 聚焦：border-focus + 轻微 glow

### 2.3 辅助说明

```
💡 提示文案
```

- 字号：12px
- 颜色：text-secondary
- 图标：小灯泡或信息图标

### 2.4 错误提示

```
❌ 错误文案
```

- 字号：12px
- 颜色：state-error
- 图标：伤心小熊或小叉号
- 动效：输入框轻微摇晃（-3° ~ 3°，400ms）

## 3. 表单状态

### 3.1 初始状态

- 所有字段为空
- 提交按钮禁用（或允许点击后校验）

### 3.2 填写中

- 实时校验（可选）
- 字段完成标记（可选，小对勾）

### 3.3 提交中

- 按钮变为加载状态
- 文案："小信使正在送信..."
- 禁用所有字段

### 3.4 成功状态

- 显示成功 Toast
- 文案："回忆已收藏进故事书~"
- 跳转或清空表单

### 3.5 错误状态

- 显示错误字段
- 滚动到第一个错误字段
- 错误提示动画

## 4. 特殊表单

```

```

## 5. 情绪化文案

| 场景 | 系统文案 | Fairy 文案 |
|------|----------|------------|
| 必填 | 此项必填 | 这里需要填写哦~ |
| 邮箱格式 | 邮箱格式不正确 | 小信使找不到这个地址呢 |
| 密码短 | 密码不能少于6位 | 我们的暗号太短啦 |
| 密码弱 | 密码强度不足 | 这个暗号太简单啦 |
| 提交中 | 提交中... | 小信使正在送信... |
| 提交成功 | 提交成功 | 信件已安全送达~ |

---

## 6. AI 生成规则

1. 表单字段必须有标签，禁止 placeholder 替代标签
2. 错误提示必须情绪化，禁止 "Error: ..."
3. 提交按钮必须有加载状态
4. 长表单必须分步或分组
5. 键盘弹出时必须调整布局，避免遮挡
