# Fairy Token Reference

## Color Tokens

Primary:

- fairyPink
- roseDust

Neutral:

- creamPaper
- moonWhite

Text:

- cocoaBrown
- softCocoa

Accent:

- amberGold

## Spacing Tokens

Use 4px base scale.

- xs: 4
- sm: 8
- md: 12
- lg: 16
- xl: 24
- xxl: 32
- xxxl: 48

## Radius Tokens

- small: 8
- medium: 16
- large: 24
- fairy: 32

## Shadow Tokens

Paper shadow:

Used for cards and containers.

Floating shadow:

Used for important actions.

## Code Rule

Prefer theme tokens.

Avoid hardcoded values:

```js
color: '#EFA7B8'
```

Prefer:

```js
colors.fairyPink
```
