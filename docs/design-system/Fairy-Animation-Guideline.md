# Fairy 动效指南

## 原则

动效应当营造温暖感和情感连接。

避免：

- 过度特效
- 过快的移动
- 没有明确目的的装饰性动画

## 按钮交互

推荐流程：

按下
-> 轻微缩小
-> 弹性回弹
-> 可选的爱心或星光反馈

## 弹窗动效

打开：

- opacity 从 0 到 1
- scale 从 0.92 到 1
- translateY 从 20 到 0

关闭：

- 淡出
- 轻柔缩小

## 页面转场

推荐样式：

- 书页翻动
- 淡入淡出
- 纸张滑动

## AI 处理动效

用于 AI 创作页面时：

使用：

- 魔法棒感
- 星光闪烁
- 有故事感的进度表达

示例：

"正在写入你的童话故事"

而不是：

"Loading..."

## 实现

推荐技术：

- react-native-reanimated
- expo-haptics

通过共享组件保持动效可复用。
