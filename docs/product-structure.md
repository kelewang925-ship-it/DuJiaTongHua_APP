# 独家童话 Product Structure

## Core Tabs

### 1. Home / 记录中心

Purpose: daily record hub and emotional entry.

Content:
- Today quick record entrance
- Recent mixed story stream
- Quick create: diary, photo, AI creation
- Love day counter
- Soft storybook atmosphere

### 2. Couple Space / 情侣空间

Purpose: private shared timeline.

Content:
- Two-person profile card
- Couple timeline
- Likes and comments later
- Love day counter
- Shared memory highlights

### 3. Fairy Workshop / 童话工坊

Purpose: emotional AI creation center.

Content:
- AI comic generation entrance
- AI memory video entrance
- Creation history
- Generation progress later

### 4. Mine / 我的

Purpose: private collection and settings.

Content:
- Profile
- Anniversary management
- Export and backup
- Drafts
- Settings

## Page Flow

Login -> Check couple binding

If not bound:
Login -> Invite or accept invitation -> Binding success -> Main tabs

If bound:
Login -> Main tabs

Main tabs:
- Home
- Couple Space
- Fairy Workshop
- Mine

Each feature module uses an independent page stack.
