# 《独家童话》image2 页面效果图生成提示词

> 用途：将当前项目 `docs/` 文档与 `assets/design/*.png` 效果图重新整理为 image2 可执行的页面效果图生成任务。
>
> 核心要求：**一个界面 = 一张图**。每次只生成一个页面效果图，不要把多个页面、多个手机屏幕、多个流程步骤合并到同一张图片里。

---

## 1. 全局生成规则（必须严格遵守）

你现在是《独家童话》APP 的高保真 UI 设计生成助手，需要根据项目文档和已有 PNG 效果图/示意图，重新生成移动端页面效果图。

### 1.1 一图一页规则

- **一个界面只生成一张图片。**
- **一张图片里只允许出现一个完整页面效果图。**
- 不要把多个页面、多个手机屏幕、多个流程、多个状态拼接成一张长图或九宫格。
- 不要输出 App 全流程总览图，不要输出多页面海报，不要输出多屏并排展示图。
- 如果一个功能有多个页面，例如“登录页、情侣邀请页、情侣绑定确认页”，必须分别生成三张图片。
- 如果一个页面有弹窗、底部浮层、空状态、加载状态等必要组件效果，可以在**该页面内部**以当前页面状态的形式呈现，但不要额外并排放其他页面。
- 每张图的主体必须是一个移动端手机页面，高保真、可用于后续开发对照。

### 1.2 页面完整性规则

每张页面效果图必须包含该页面所需的所有核心内容：

- 页面顶部：标题、副标题、返回按钮、右侧操作按钮等。
- 页面主体：卡片、列表、表单、记录流、插画、统计信息、状态标签等。
- 页面底部：主 Tab 页面需要包含底部 TabBar；二级页面按实际需要包含底部主按钮、浮动按钮或安全区留白。
- 页面组件效果：按钮、输入框、标签、卡片、分割线、空状态、Toast、Dialog、BottomSheet、ActionSheet 等，如果该页面场景需要，就在当前页面中体现。
- 页面视觉氛围：童话绘本、奶油纸感、轻手绘、贴纸感、柔和治愈、留白充足。

### 1.3 禁止项

- 不要生成多个页面合并图。
- 不要生成网页版后台界面。
- 不要生成真实照片风。
- 不要生成赛博蓝紫、霓虹科技风、冷冰冰 AI 工具风。
- 不要使用高饱和粉色。
- 不要低幼卡通，不要过度儿童化。
- 不要复杂背景，不要大面积噪点，不要抢主体。
- 不要出现品牌 Logo。
- 不要出现不可控真实文字水印。
- 不要把 PNG 效果图中的多个小屏原样做成多屏拼图；PNG 效果图只是结构参考，最终必须拆成单页面高保真图。

## 2. 统一视觉规范

### 2.1 产品定位

产品名称：**独家童话**。

核心概念：把情侣日常生活变成一本持续生长的私人故事书。

关键词：

- 情侣情感记录
- 私密双人空间
- 童话绘本
- 奶油纸感
- 桃粉、干玫瑰、可可棕、琥珀金
- 手绘贴纸式组件
- 回忆收藏
- 温柔 AI 魔法感

### 2.2 色彩

请严格使用以下色彩气质：

- 背景色 / 月白：`#F8F6F2`
- 卡片色：`#FFF9F4`
- 桃粉卡片：`#FFF0F2`
- 主色 / 桃夭：`#F6BEC8`
- 主色深色：`#DFA0AC`
- 辅助色 / 藕丝秋半：`#E9D7D2`
- 强调色 / 枯玫瑰：`#B08A8F`
- 正文色 / 可可棕：`#6B4F4F`
- 柔和文本色：`#9B7A7A`
- 金色 / 琥珀金：`#D8B384`

### 2.3 页面质感

- 背景使用月白奶油纸感，有非常轻的纸张纹理。
- 卡片使用暖白纸张质感，圆角柔和，轻描边，轻阴影。
- 装饰元素使用小爱心、星星、小花、纸胶带、拍立得角标、书签、丝带、魔法棒。
- 图标要像轻手绘贴纸，不要像系统默认 icon。
- AI 页面可以有少量星点、墨水显影、魔法棒、金色微光，但必须温柔，不要赛博。
- 情侣空间强调私密双人故事，不要公开社交动态流。

### 2.4 通用组件风格

所有页面都应保持以下组件语言：

- `FairyPage`：统一月白纸感背景、顶部留白、底部安全区。
- `FairyHeader`：eyebrow、标题、副标题、返回按钮、右侧操作。
- `FairyCard`：大圆角纸感卡片、轻描边、柔和阴影。
- `FairyButton`：主按钮桃粉，次按钮暖纸色或藕粉色。
- `FairyTag`：小胶囊标签，用于心情、类型、AI 状态、会员状态。
- `FairyInput`：圆角输入框，label、图标、辅助说明、错误态。
- `FairyEmptyState`：插画、标题、说明、按钮。
- `FairyDialog` / `FairyBottomSheet`：贴纸式弹窗或底部浮层。
- `FairyTabBar`：贴纸式底部 Tab，激活态桃粉胶囊，图标轻微浮起。
- `MemoryCard`：日记、照片、漫画、纪念日、视频记录卡。
- `MemoryWall`：首页回忆碎片墙，不规则拼贴卡片。
- `CoupleTimeline`：情侣空间手绘时间轴。
- `WorkshopCard`：AI 漫画、AI 视频、文本转漫画入口卡片。

## 3. image2 输出规格

- 输出比例：移动端竖版页面效果图，建议 `9:19.5` 或接近 iPhone 页面比例。
- 输出内容：一张完整手机页面截图式 UI，不要带外部展示框，不要做样机 mockup。
- 分辨率：高清，建议宽度不低于 1080px。
- 风格：高保真移动端 UI 设计稿，适合交给前端照图实现。
- 每次生成只处理一个页面。
- 图片内可以出现少量中文 UI 文案，但必须自然、清晰、温柔，不要出现乱码。
