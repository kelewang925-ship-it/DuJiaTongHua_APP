## 2. 全项目统一图片风格

所有图片必须保持统一风格：

```text
童话绘本风格，奶油纸质感，柔和桃粉、干玫瑰、可可棕、琥珀金点缀，轻手绘线条，贴纸感，温柔治愈，留白充足，适合情侣情感记录 App，不要高饱和，不要科技蓝紫风，不要真实照片风，不要低幼卡通，不要复杂背景，不要出现真实文字，不要出现品牌 Logo。
```

推荐画面气质：

- 像一本恋爱故事书里的小插画；
- 有纸张、胶带、拍立得、小花、星星、爱心、魔法棒等轻元素；
- 画面要适合移动端 UI 卡片，不要太满；
- 不要把角色画得过于写实；
- 人物可以是抽象情侣形象、背影、剪影、头像贴纸，不需要真实五官。

## 3. 输出规格

```text
尺寸：1200 × 900
格式：PNG 或 WebP
背景：奶油纸感或透明背景均可，优先奶油纸感浅背景
用途：首页、情侣空间、童话工坊、纪念日等 Hero 区域
```

```text

格式：PNG 或 WebP
背景：透明或浅奶油纸背景
用途：无日记、无照片、无搜索结果、无通知等
```

```text
尺寸：512 × 512
格式：PNG
背景：透明
用途：爱心、星星、小花、胶带、魔法棒、信封、拍立得等装饰
```

```text
尺寸：1080 × 1440 或 1600 × 2260
格式：PNG 或 WebP
用途：PDF封面、分享预览、纪念册封面
```

## 10. 图片生成后的项目接入步骤

图片生成完成后，把图片放入：

```text
assets/images/illustrations/home-cover-v1.png
assets/images/illustrations/couple-space-cover-v1.png
assets/images/illustrations/workshop-cover-v1.png
```

然后修改：

```text
src/components/FairyImage.js
```

打开映射：

```js
const imageSourceMap = {
  homeCover: require('../../assets/images/illustrations/home-cover-v1.png'),
  coupleCover: require('../../assets/images/illustrations/couple-space-cover-v1.png'),
  workshopCover: require('../../assets/images/illustrations/workshop-cover-v1.png'),
};
```

并确保真实图片渲染时使用：

```js
<Image source={source} resizeMode="contain" style={styles.image} />
```

保留 fallback，避免图片缺失导致页面崩溃。
