## 建议测试顺序

1. 依次打开文本转漫画、漫画配置、照片转漫画和短视频配置，检查来源、画风、分镜、时长与开关状态。
2. 分别触发漫画和视频生成，确认进入 `/ai/progress` 后目标作品类型正确。
3. 打开 `/ai/video-preview`，检查封面、字幕输入、音乐、片段选择、保存与分享反馈。
4. 打开 `/ai/history`，切换全部/漫画/视频/生成中筛选，检查分享、进度跳转与空状态。

## Android 重点检查

- Safe Area：长表单顶部标题与底部操作不被挖孔或手势条遮挡。
- 键盘：字幕和自由文本输入时页面可滚动，保存按钮不被键盘挡住。
- 图片：正式绘本预览、视频封面和历史缩略图无拉伸、模糊或过大纹理内存压力。
- 字体与换行：360–430dp 宽度下标签、分镜、时长和按钮文案不截断。
- 阴影与模糊：卡片 elevation、顶部滚动层级和 FairyPage 阴影无灰色圆弧。
- 触控：筛选、图片选择、Switch、分享与返回按钮点击区域稳定。

## Web 无法确认

- Android 键盘 resize/pan 行为、原生返回键与手势返回。
- elevation、BlurView、图标字体首次加载和真实 GPU 图片纹理表现。
