# 阶段 3 Android 人工验证清单

> 页面代码与 Web/Android export 已检查；Android 原生效果待人工验证。

## 本阶段页面

1. `/anniversary`
2. `/anniversary/edit`
3. `/anniversary/template`
4. `/data/pdf-export`
5. `/data/export-preview`
6. `/data/storage`

## 建议测试顺序

1. 从 `/anniversary` 筛选分类、进入编辑页新增纪念日，再编辑刚添加的章节。
2. 在 `/anniversary/template` 切换 5 套模板和预览问题，使用模板进入编辑页。
3. 在 `/data/pdf-export` 切换范围、内容、封面、纸张和清晰度并生成预览。
4. 在 `/data/export-preview` 切换 4 类页面缩略图，检查返回修改、分享和确认导出状态。
5. 在 `/data/storage` 选择空间项，仅清理临时缓存并确认照片、作品和导出数据仍保留。

## Android 重点检查

- 标准头部：数据模块三页的返回、居中标题、右侧操作、滚动固定层级与正文避让。
- Safe Area：挖孔屏顶部固定头部和底部按钮不被遮挡。
- 响应式：360–430dp 下纪念日模板双列、封面选择和导出页缩略图不横向溢出。
- 键盘：纪念日名称、日期和备注输入时表单可滚动且保存按钮可达。
- 图片：多张正式封面缩略图在列表与大预览中清晰、比例正确。
- 弹窗与触控：缓存清理确认、Switch、筛选、模板和封面点击不误触。
- 字体：页数、文件大小、日期和较长标题不与头部两侧操作重叠。

## Web 无法确认

- Android 顶部 inset、原生 elevation/blur、系统返回键和键盘行为。
- 大量封面同时解码时的真实设备内存与滚动流畅度。
